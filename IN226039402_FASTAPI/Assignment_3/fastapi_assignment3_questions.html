# Add this ABOVE @app.get('/products/{product_id}') @app.get('/products/audit') def product_audit(): in_stock_list = [p for p in products if p['in_stock']] out_stock_list = [p for p in products if not p['in_stock']] stock_value = sum(p['price'] * 10 for p in in_stock_list) priciest = max(products, key=lambda p: p['price']) return { 'total_products': len(products), 'in_stock_count': len(in_stock_list), 'out_of_stock_names': [p['name'] for p in out_stock_list], 'total_stock_value': stock_value, 'most_expensive': {'name': priciest['name'], 'price': priciest['price']}, }

Apply a Category-Wide Discount

Scenario: It's a sale. All Electronics products should get a 10% price cut in one go — not one by one. Build PUT /products/discount that takes a category name and a discount percentage and updates every product in that category at once.
Your Tasks
🔧

Add PUT /products/discount with query params: category (str, required) and discount_percent (int, required, 1–99). Place it above /products/{product_id}

💸

Loop through all products, find ones matching the category, apply: new_price = int(price × (1 - discount_percent / 100))

📊

Return how many products were updated and show their new prices. If no products found in that category, return a friendly message

Test These
PUT/products/discount?category=Electronics&discount_percent=10→ all Electronics get 10% off
GET/products→ Electronics prices are reduced
PUT/products/discount?category=Furniture&discount_percent=20→ no products found message
💡 Show Hint
Hint — Add above /products/{product_id}
@app.put('/products/discount') def bulk_discount( category: str = Query(..., description='Category to discount'), discount_percent: int = Query(..., ge=1, le=99, description='% off'), ): updated = [] for p in products: if p['category'] == category: p['price'] = int(p['price'] * (1 - discount_percent / 100)) updated.append(p) if not updated: return {'message': f'No products found in category: {category}'} return { 'message': f'{discount_percent}% discount applied to {category}', 'updated_count': len(updated), 'updated_products': updated, }
